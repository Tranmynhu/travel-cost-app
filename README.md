# Travel Cost App
This is a simple **JavaFX** application that calculates the total cost of road trip.

## Features
- Enter the Miles to Drive, Mile per Gallon, Gasoline Price, Parking Cost, and Tolls Cost.
- Click **Calculate** to see the total cost of your trip.
- Click **Clear** to reset all input fields.
- User-freindly interface built with JavaFX.

## Technologies Used
- Java for core programming language.
- JavaFX for building the graphical user interface.
